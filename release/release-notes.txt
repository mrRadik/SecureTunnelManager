Secure Tunnel Manager 1.2.4
Изменения с версии 1.1.0 (все незакоммиченные правки)

═══════════════════════════════════════
АВТООБНОВЛЕНИЕ
═══════════════════════════════════════

• Надёжный запуск MSI-установщика — msiexec.exe стартует напрямую с правами администратора (UAC), без промежуточного cmd.exe
• После автообновления приложение автоматически перезапускается — без галочек и дополнительных диалогов
• Упрощённый процесс установки: подтверждение обновления → скачивание → одно финальное подтверждение с текстом о перезапуске
• Защита от повторного запуска установки — блокировка параллельных попыток обновления
• Корректный возврат на UI-поток после скачивания MSI (диалоги подтверждения отображаются стабильно)
• Расширенное логирование: ошибки скачивания, запуска установщика, отмена UAC пользователем
• Сохранение release notes перед установкой — для окна «Что нового» после перезапуска

Скачивание и проверка:
• Загрузка во временный файл (.download) с атомарным переименованием по завершении
• Проверка SHA-256 с подробным логом при несовпадении (размер файла, ожидаемый и фактический хеш)
• Предупреждение в лог при подозрительно маленьком размере загрузки
• Нормализация полей манифеста обновления (trim пробелов в version, url, sha256, releaseNotes)
• HttpClient с таймаутом 30 минут, User-Agent и Accept-заголовками для GitHub Releases

═══════════════════════════════════════
ОКНО «ЧТО НОВОГО»
═══════════════════════════════════════

• Увеличенный размер окна (680×560) — release notes читаются комфортнее
• Прокручиваемая область заметок занимает всё доступное пространство
• Release notes загружаются из GitHub: сначала latest/update.json, затем update.json конкретной версии
• Встроенные release notes в приложении (embedded resource) — доступны без интернета
• Сохранённые при автообновлении заметки (pending-release-notes.json) — приоритет при первом запуске
• Корректное сравнение версий major.minor.build — окно показывается только при реальном обновлении
• Отдельный метод сохранения LastAcknowledgedVersion без перезаписи остальных настроек

═══════════════════════════════════════
УСТАНОВЩИК (MSI / WiX)
═══════════════════════════════════════

• Автозапуск приложения после установки через WixUnelevatedShellExec — запуск от имени пользователя, не elevated
• Свойства LAUNCHAPP и AUTOMATED_UPDATE — управление перезапуском из приложения при автообновлении
• Автозапуск срабатывает при LAUNCHAPP=1 или AUTOMATED_UPDATE=1 (не при удалении)
• CloseApplication — закрытие работающего приложения перед обновлением (уже было, сохранено)
