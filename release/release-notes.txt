Обновления и установка:
• Автообновление через MSI — проверка при запуске и кнопка в настройках
• Окно «Что нового» при первом запуске после обновления
• Сборка инсталлятора генерирует update.json для GitHub Releases

Настройки:
• Секция «Туннели»: интервал реконнекта и пауза circuit breaker
• Автоблокировку хранилища можно включить или отключить
• Секция «Обновления» с текущей версией и ручной проверкой

Надёжность SSH:
• Polly (circuit breaker и повторные попытки) для SSH-подключений
• Бессрочный автоматический реконнект туннелей вместо лимита в 3 попытки
• Интервалы переподключения настраиваются в параметрах

Интерфейс:
• Иконки Segoe Fluent Icons с fallback для Windows 10
• Обновлены иконки в навигации и экране хранилища паролей

Прочее:
• Убраны опции «сворачивать в трей при запуске» и «запускать свёрнутым с Windows»
• Приложение открывается в окне при каждом старте
